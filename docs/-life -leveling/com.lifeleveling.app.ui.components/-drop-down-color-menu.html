Parameters

list of color choices

selectedIndex

Variable for storing the selected option index

onSelectedChange

What to do when an option is selected. Pass in { selectedIndex = it } for selected Index to be updated

expanded

The boolean that controls if the menu shows or not

readOnly

Controls if the inner text field can be typed into or only read

arrowSize

Changes the size of the arrow on the dropdown box

textColor

Sets the color of all the text and the arrow

backgroundMainColor

Main color of the text field. Is also used in the menu as one of the alternating colors

accentColor

The second color of the alternating colors in the menu

outlineColor

Color of the text field outline

selectedBackground

A highlight to the option that is currently selected
